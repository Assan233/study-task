## 一些命令

#### gents

`npm run gents {id} {index}`

- 通过`questionsId`读取`README`、`template.ts`、`test-cases.ts`文件自动创建 `答题ts` 文件。
- 自动在索引目录md文件里面添加对应的 **题目索引**

```js
// 创建 00059-hard-get-optional 测试文件
   npm run gents 59 1
```
